# Calculator with History

A simple Python console-based calculator that performs basic arithmetic operations and saves calculation history to a file.

This project is made for practicing Python basics such as functions, file handling, and user input.

---

## Features

- Addition, subtraction, multiplication, and division
- Saves calculation history in a text file
- View previous calculations
- Clear calculation history
- Handles division by zero

---

## Concepts Used

- Functions
- Conditional statements
- Loops
- File handling (`read`, `write`, `append`)
- String operations
- User input handling
