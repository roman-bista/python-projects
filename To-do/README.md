To-Do List Manager

A command-line To-Do List Manager built with Python that allows users to manage daily tasks with persistent JSON storage.

This project demonstrates practical backend programming concepts including:

CRUD operations
JSON data handling
file management
modular programming
task state management
Features
Add new tasks
View all tasks
Mark tasks as completed
Delete tasks
Edit existing tasks
Task priority system
Due date support
Persistent storage using JSON
Input validation
Command-line interface
