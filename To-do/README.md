# To-Do List Manager

A beginner-friendly command-line To-Do List application built with Python.

This project allows users to:
- add tasks
- view tasks
- mark tasks as completed
- save tasks permanently using JSON


## Features

- Add new tasks
- View all tasks
- Mark tasks as completed
- Persistent task storage using JSON
- Simple command-line interface
- Input validation
- File handling with JSON

## Technologies Used

- Python 3
- JSON
- OS module


## Concepts Practiced

- Functions
- Loops
- Conditions
- Lists & Dictionaries
- JSON file handling
- Exception handling
- Modular programming
- Path management using `os`
