# Fake News Headline Generator 📰

A simple Python program that generates random fake news headlines using predefined subjects, actions, and places.  
This project is designed to practice basic Python concepts and console-based interaction.

---

## 🔧 Features
- Generates random fake news headlines
- Uses user input to continue or stop
- Simple and beginner-friendly logic

---

## 🧠 Concepts Used
- `while` loop
- Conditional statements (`if`)
- `random` module
- String methods:
  - `strip()`
  - `lower()`
- User input handling

---

## ▶ How to Run

Make sure Python is installed, then run:

```bash
python fakeheadline.py
