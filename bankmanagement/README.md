# Python Bank System

Simple banking system made with Python and JSON storage.

## Features
- Create account
- Deposit money
- Withdraw money
- Delete account
- Save data in JSON file

## Technologies
- Python
- JSON
- OOP

## Run

```bash
python main.py
```
