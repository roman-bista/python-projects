# Restaurant Order System

Simple restaurant ordering program made with Python.

## Features

- Display food menu
- Order items
- Calculate total bill
- Add multiple items
- Input checking

## Technologies

- Python
- Dictionary
- Conditions

## Run

```bash
python main.py
```
