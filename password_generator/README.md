Random Password Generator

A beginner-friendly Python project that generates secure random passwords based on user preferences.

Users can choose:

password length
uppercase letters
digits
special characters

The program then generates a strong random password.

Features:

Random password generation
Custom password length
Optional uppercase letters
Optional digits
Optional special characters
Input validation
Play again option
Beginner-friendly code structure
